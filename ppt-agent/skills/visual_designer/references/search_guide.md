# 搜索规范指南

本文档是 visual_designer 的补充参考，按需查阅。

## 搜索原则

当内容空洞、缺少细节或涉及具体数据时，可调用 search 工具搜索真实信息，然后基于搜索结果填充内容。

## 搜索规范（必须严格遵守）

- 每次搜索**只传入一个核心关键词**，不要拼接多个关键词
- 关键词要求：简洁、精准、长约 2-5 个词
- 如果需要搜索多个不同主题，**必须分多次调用** search 工具
- 示例：
  - 正确：`{"query": "深度学习发展历程"}`
  - 错误：`{"query": "深度学习 发展历程 里程碑"}`

## 分页策略

如数据/要点超过当前布局容量，**主动拆分为多页**：

- 同一主题多个小点需详细解释 → **每点一页**，大标题不变副标题变
- 单页要点超过6条 → 拆分
- 几何体内文字估算后超出边界 → 拆分或精简内容
- 数据对比项过多 → 拆分专题页
